# WITO Smart Health

Dashboard for intraoral and biometric monitoring.

## Features

- Patient and Provider views
- SmartAligner integration
- Apple HealthKit integration
- Real-time biometric data visualization
- AI-powered health insights (Gemini)

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Google Gemini API
- Apple HealthKit

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file with:

```
GEMINI_API_KEY=your_api_key
```
